# Blackjack Game (JavaScript)

## Project Overview
This project is a simple browser-based **Blackjack game** built using **HTML, CSS, and JavaScript**.  
It demonstrates the core rules of Blackjack and helps practice JavaScript concepts such as arrays, functions, loops, conditional statements, and DOM manipulation.

## Objectives
- Implement basic Blackjack game logic  
- Practice JavaScript programming concepts  
- Understand DOM manipulation and event handling  
- Build an interactive web-based game  

## Technologies Used
- **HTML5** – Game structure  
- **CSS3** – Styling and layout  
- **JavaScript (ES6)** – Game logic and interactivity  

## Game Features
- Start a new Blackjack game  
- Draw new cards dynamically  
- Automatic calculation of card sum  
- Display win, lose, or continue game messages  
- Player name and chips display  
- Button-based interaction  

## Game Rules Implemented
- Cards are randomly generated between **1 and 13**  
- Card values:  
  - Ace (`1`) → **11**  
  - Face cards (`11–13`) → **10**  
  - Other cards → face value  
- Player wins if the sum equals **21**  
- Player loses if the sum exceeds **21**  
- Player can draw new cards while still alive
